Файл: Konfiguration.sys
1 строка: количество уровеней
2 строка: длина заставки
3 строка: высота заставки
последующие: заставка

Файл: Niveau xxx.dat
1 строка: высота уровня
последующие: топография уровня
0 - проходимая область
1 - непроходимая область
2 - игрок
3 - монстр (горизонтальный)
4 - монстр (вертикальный)
9 - пункт назначения

Заметка: номера уровней должны идти последовательно.
Заметка: присутствие монстра обязательно и только в ОДНОМ экземпляре.
Заметка: в таблице рекордов (Rekorde.tab) заранее должно быть наличие рекордов на каждый уровень.
